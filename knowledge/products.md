# 大雁的产品和项目

## ContentFlow — 自媒体内容工作台
**定位**：AI 自媒体内容工作台，帮助内容创作者把素材自动生成小红书 + 公众号双平台文案。

**核心功能**：
- 粘贴素材 → 自动生成小红书爆款文案（含 AI 去痕迹化处理）
- AI 智能配图匹配（3 张统一风格原创插图）
- 双平台适配：一份输入，同时输出小红书版 + 公众号版
- 风格学习记忆：越用越像你自己

**适合谁**：自媒体博主 / 知识 IP 创作者 / 每天发内容的个人品牌运营者

**状态**：开发中（core features 已完成，持续迭代）

## drama-clip — 短剧二创桌面工具
**定位**：短剧二创自动化流水线桌面工具。

**核心功能**：
- 扫描本地剧集目录，AI 自动生成分镜脚本
- TTS 配音 + 字幕对齐
- 批量渲染导出
- 本地运行，无需云端，数据不上传

**技术栈**：Electron + Python + TTS + AI

**状态**：WIP，内部使用中

---

## 看图识字 — AI 个性化识字绘本小程序
**定位**：基于孩子当前识字量智能生成专属绘本的微信小程序。

**核心功能**：
- 基于孩子的识字量生成专属绘本，每本新增 3-5 个生字
- 大字配拼音，支持跟读（TTS 配音）
- 绘本角色可按孩子喜好定制
- 科学分级，自动进阶

**适合谁**：3-8 岁孩子的家长，希望用个性化内容提升识字效率

**状态**：WIP，开发中

---

## 电子绘本 — AI 辅助实体绘本生成工具
**定位**：输入主题和角色，AI 完成全流程，最终输出可印刷的高清 PDF。

**全流程**：构思 → 分镜 → 生图 → 排版 → 封面

**核心价值**：生成一本真正属于孩子的实体绘本，可以送去印刷成真书。

**状态**：WIP，探索阶段
